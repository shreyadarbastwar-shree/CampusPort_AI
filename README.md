# CampusPort

CampusPort is a web-based platform where students can upload their projects and explore other student's projects.

## 🔧 Technologies Used
- PHP
- HTML
- CSS

## ✨ Features
- Student registration & login
- Project upload system
- Project viewing dashboard
- Manage student projects

## 🚀 How to Run
1. XAMPP/WAMP start karo
2. Project folder htdocs me daalo
3. Browser me open karo: localhost/CampusPort
